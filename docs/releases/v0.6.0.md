# Kimi-GUI 0.6.0

Version 0.6.0 makes the desktop workflow easier to understand and easier to
redirect while Kimi is working. It also adds explicit update consent and a
single, guided path for connecting Kimi Code CLI.

## Workspace-aware new conversations

The new-chat composer now restores the most recently used project directory.
Before the first message, you can choose another directory and select an
existing local Git branch. Repositories without a selectable branch show
**None** in a disabled selector.

![New conversation project and branch controls](https://raw.githubusercontent.com/kaminion/Kimi-GUI/v0.6.0/docs/media/release-0.6.0-new-chat.png)

## File changes where you need them

The changed-file count and cumulative added/deleted lines now sit directly
above the prompt. The former summary position shows the current working branch,
and Swarm always displays an explicit **ON** or **OFF** state.

![Composer change summary, branch, and Swarm state](https://raw.githubusercontent.com/kaminion/Kimi-GUI/v0.6.0/docs/media/release-0.6.0-composer.png)

Selecting the summary opens the existing right-side panel on its **Changes**
tab. Activity and change review continue to share one panel, so opening file
details never stacks another inspector.

![Tabbed file change review panel](https://raw.githubusercontent.com/kaminion/Kimi-GUI/v0.6.0/docs/media/release-0.6.0-changes-panel.png)

## Steer work without stopping it

The prompt stays editable during an active run. Press Enter to inject an
adjustment into the current work; stopping remains a separate action. Both the
CLI engine and the built-in direct engine support steering, and direct sessions
persist each adjustment in conversation history.

![In-turn work adjustment with separate stop control](https://raw.githubusercontent.com/kaminion/Kimi-GUI/v0.6.0/docs/media/release-0.6.0-steer.png)

## Clear lifecycle decisions

When the built-in engine starts, one optional dialog explains CLI agent mode.
It offers **Connect CLI** when Kimi Code CLI is ready, **Install CLI** when it is
missing, and a **Do not show again** option.

![Optional CLI agent mode connection dialog](https://raw.githubusercontent.com/kaminion/Kimi-GUI/v0.6.0/docs/media/release-0.6.0-cli-prompt.png)

New app versions are no longer downloaded silently. The app asks first, shows
download progress in the same dialog, and asks again before restarting to
install.

![Update consent dialog](https://raw.githubusercontent.com/kaminion/Kimi-GUI/v0.6.0/docs/media/release-0.6.0-update-prompt.png)

## Additional improvements

- Reworked code-block language and copy controls for a predictable toolbar.
- Kept long Thinking/Working sections easy to collapse with a sticky header.
- Increased separation between the conversation section label and title.
- Preserved a single tabbed agent panel for Activity and Changes.
- Added safer, shell-free Git branch discovery and switching.
- Kept branch switching non-destructive: Git refuses unsafe switches instead
  of overwriting local work.

## Downloads

- macOS Apple Silicon: DMG or ZIP
- macOS Intel: DMG or ZIP
- Windows x64: NSIS installer or portable executable

Development builds are unsigned. macOS may show a Gatekeeper warning, and
automatic installation requires a properly signed production build.
